Secured by itsyouonline with scopes:
  • user:name

OAuth2 is a protocol that lets external apps request authorization to private details in a user's account without getting their password.

  • Request
  • Response
  • Security

URI Parameters

  • namespaceid: required (string)

Query Parameters

  • from: required (integer)

    Unix timestamp from when the desired log entries returned should start

  • to: (integer)

    Unix timestamp until when the desired log entries returned should end if not specified, returns the log entries until now

HTTP status code 200

Log entries successfully returned

Body

Media type: application/json

Type: array of TlogEntry

Items: TlogEntry

  • header: required (object)
    • timestamp: required (integer)

      Unix timestamp when this entry has been created

    • hash: required (string)

      Hash of the data. Used to validate integrity

  • data: required (string)

    Array of bytes

HTTP status code 400

Bad request, from is bigger then to
